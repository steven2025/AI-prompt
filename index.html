DeepSeek 增强助手 · 使用说明

建议先安装到桌面

打开网页后，如果出现“安装 DeepSeek 增强助手”的提示，建议点击安装。安装到电脑或平板桌面后，可以像普通应用一样直接打开，使用更方便；如果已经安装，就不会重复提示。

这个工具解决什么问题？

  • 普通 DeepSeek 对话过长时，编程容易中断。
  • 修改程序时，常常只返回修改片段，不利于零基础学生直接使用。
  • 生成的结果缺少统一预览、文件查看和下载入口。

第一次使用

  1. 点击右上角 ⚙ ，填写你自己的 DeepSeek API Key。
  2. Base URL 一般保持默认：https://api.deepseek.com。
  3. 模型建议先选 deepseek-chat。
  4. 点击“测试连接”，成功后即可开始使用。

基本操作

  1. 打开项目：优先使用“打开前端项目”和“打开云函数包”，浏览器授权后可自动写回原文件。
  2. 修改程序：点击左侧文件树中的文件，在中间代码区编辑，右侧 AI 助手围绕当前文件对话。
  3. 查看结果：中间“预览”可查看 HTML；“代码”可查看前端、配置和云函数文件。
  4. 下载：点击“下载主文件”或“下载全部”。

右侧“培训”按钮

点击右侧竖向 培训 按钮，可打开 AI 开发培训小程序，用于学习术语、提示词和课程内容。

推荐提示词

请检查当前前端程序的结构、样式、交互和移动端适配。

请检查前端接入大模型、对象存储、云函数接口和环境变量配置是否完整。

请根据当前云函数包生成环境变量清单，并说明 GitHub 发布注意事项。

请根据我粘贴的报错定位问题文件，并给出可直接替换的修改片段。

长时间连续修改后，可点击“整理上下文”再继续。
